# Basil Mirza — Developer Portfolio

This is my personal developer portfolio website showcasing my projects, skills, education, and experience as a software engineer.

The portfolio highlights several full-stack and AI-based projects including developer tools, machine learning applications, and SaaS-style platforms.

## Live Website

https://basilmirza.vercel.app/

## Tech Stack

Frontend
- Next.js
- React
- Tailwind CSS
- TypeScript / JavaScript

Deployment
- Vercel

## Features

- Responsive modern portfolio design
- Projects showcase
- Skills section
- Education and experience
- Resume download
- Contact section

## Featured Projects

### RepoLens AI
AI-powered GitHub repository analysis tool that detects tech stack, architecture insights, and quality signals.
